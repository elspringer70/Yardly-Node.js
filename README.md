# Yardly

A backyard rental website for dogs, inspired by Airbnb and Sniffspot.

## Overview

Yardly is a platform that connects dog owners with homeowners who have fenced backyards. Dog owners can rent space by the hour for their dogs to play, while homeowners earn money sharing their yard.

## Features

- **Browse Backyards**: Search and filter backyards by location, size, and amenities
- **Book a Spot**: Reserve a backyard for your dog with easy booking
- **Host Your Yard**: List your fenced backyard and earn money
- **Secure Payments**: Safe and reliable payment processing
- **Ratings & Reviews**: See what other users say about each backyard

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive design with mobile-first approach

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process required - just open and run!
